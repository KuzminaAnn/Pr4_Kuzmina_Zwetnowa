Для написания программы была выбранна среда программирования "Microsoft Visual Studio" как самая адаптивная среда из имеющихся 
для выбранного нами языка программирования. Разработка происходила в приложении "WPFApp" для болеее отзывчивого и удобного пользовательского интерфейса.
Программа была написана на C# с подключенной библиотекой Math для облегчения процесса написания уравнений в коде.

В основе нашего приложения лежат 4 страницы из которых одна главная навигационная, в которую переходят все другие побочные страницы и откуда
происходит переход в остальные. Три оставшиеся страницы содержат в себе примеры и формы для ввода данных, способствующих решению этих примеров.
Каждая побочная страница имеет общее оформление, кнопки для перемещения, кнопки обновления переменных или по другому очистки данныих 
и необходимые в некоторых полях подсказки, указывающие пользователю на исключения. Для удобства работы с файлами в приложении имеются папки разделяющие данные
на логиечские группы и одна страница "Class1.cs" содержащая в себе вспомогательную переменную для корректного вывода ответа на страницах.

Комментарий:график на 4 странице не запускается и соответственно данные в ячейку ответ не выводяться, по неизвестным нам причинам происходит вечный рассчет координат
(мы предполагаем, что дело в прерывном графике из-за тангенса)
